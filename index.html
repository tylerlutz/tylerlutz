What I'm Doing

I love what I do. I take great pride in what I do. And I can't do something halfway. If I do something, I do it the right way.

1. Web Development

I can develop and manage small to large web applications. Front end development work in HTML, CSS, JavaScript, jQuery and AngularJS and back end work done in ASP.NET MVC, and NodeJS. Project examples can be found on my GitHub page.

2. Mobile App Development

I have developed native mobile applications for both Android and iOS. Android development work was mostly done using Android Studio and Java, and most iOS development work has been done using Xcode and Swift. Project examples can be found on my GitHub page.

3. Systems Administration

I have been involved in many systems administration roles from end user support to implementing large projects into remote datacenters. Most systems admin work has been done in the Windows environment, but I do contain knowledge of OS X and Linux administration.

4. Database Design

I have designed databases mostly in SQL Server and MySQL. This involves the actual development of the database and the administration of the database.

5. Virtualization

Virtualization is a hobby of mine. I have worked with multiple hypervisors including WMware ESXI and Microsoft’s Hyper-V in complex lab environments. I have experience using VMware’s vSphere to manage multiple host and data stores.

6. Project Management

Project management skills were gained while attending the University of Louisville. I have managed multiple projects at once, and was the co-manager of a large 6 month technical project.
